# 🚀 Nebula Invaders - Space Invaders Futurista

Um jogo de **Space Invaders** com estilo futurista e cyberpunk, desenvolvido em HTML5, CSS e JavaScript puro.

## 🎮 Como Jogar

- **↑ ↓ ← →** : Mover a nave espacial
- **ESPAÇO** : Atirar nos invasores

## 🌌 Features

- **Estética futurista** com cores neon (azul e rosa)
- **Sistema de pontuação** e vidas
- **Níveis progressivos** (fica cada vez mais difícil)
- **Efeitos de explosão** quando destróis invasores
- **Tela de início e game over** com animações
- **Responsivo** e otimizado para browser

## 🚀 Tecnologias

- HTML5 Canvas
- CSS3 (animações e estilos)
- JavaScript puro
- Sem frameworks ou bibliotecas externas

## 🎯 Como Executar Localmente

1. **Clona o repositório:**
   ```bash
   git clone https://github.com/mac-2012-bot/space-invaders-futurista.git
   cd space-invaders-futurista
   ```

2. **Abre o ficheiro `index.html` no teu browser:**
   ```bash
   open index.html
   ```
   ou simplesmente faz duplo-clique no ficheiro.

## 🛠️ Personalização

Queres modificar o jogo? É fácil:

- **Muda as cores:** Procura por `linear-gradient` no CSS
- **Ajusta a dificuldade:** Modifica `gameState.invaderSpeed` no JavaScript
- **Adiciona novos níveis:** Expande a função `levelUp()`
